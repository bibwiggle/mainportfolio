# Junu.Lee

Personal portfolio built with Next.js. Covers kinetic sculpture, interactive installation, custom electronics, VJing, and 3D work.

**Live:** [junu.design](https://junu.design)

## Projects

- **Dancing Dancheong** — three-axis kinetic sculpture reinterpreting traditional Korean dancheong patterns through motion and robotics
- **Doki Helmet** — fully mechanical helmet with four-bar linkage and bevel gear systems, inspired by Dokkaebi folklore
- **Spiritual Mercury** — hubless ring kinetic sculpture driven by a NEMA17, developed in Blender and Fusion 360
- **Beat Bowl** — four-sided MIDI controller built for four simultaneous players, running Ableton Live via Arduino Pro Micro
- **Visual Works** — real-time visual work made in Resolume, TouchDesigner, and Blender

## Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- Lottie React

## Run locally

```bash
npm install
npm run dev
```
